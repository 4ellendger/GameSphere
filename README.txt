Тестовое задание.
Для запуска на WindowsXP требует msvcr100d.dll

Управление в TestSphere\Debug\TestSphere.exe следующее:

Держа ЛКМ - вращение вокруг центра координат
Держа ПКМ - приближение/отдаление от центра

Курсорные клавиши - смещение центра координат
NUm4,5,6,8 - вращение вокруг центра координат (идентично ЛКМ)
Home, End - приближение/отдаление от центра координат (идентично ПКМ)

Пробел (Space) - переключение камеры на объекты (по очереди)
