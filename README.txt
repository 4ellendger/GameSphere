Тестовое задание.
Для запуска на WIndowsXP требует msvcr100d.dll

Управление в TestSphere\Debug\TestSphere.exe следующее:

Держа ЛКМ - вращение вокруг центра
Держа ЛКМ - приближение/отдаление от центра

Курсорные клавиши - смещение центра
NUm4,5,6,8 - вращение вокруг центра (идентично ЛКМ)
Home, End - приближение/отдаление от центра (идентично ПКМ)

Пробел (Space) - переключение камеру на объекты по очереди
